How to install Dwave Ocean SDK

1/Create a Dwave Leap account
https://cloud.dwavesys.com/leap
Write down the API endpoint and the token (required in later stage)

2/ install the Dwave Ocean SDK
https://docs.ocean.dwavesys.com/en/stable/overview/install.html
	Create a Python env
	Command : pip install dwave-ocean-sdk
	Command : dwave setup
	Use Solver : DW_2000Q_6
	Command : dwave ping
